Lighthouse

Lighthouse is a simple jquery plugin that allows content to expand or adapt to the user's needs. It is very basic and simple in order to allow maximum compatibility and customizability. Feel free to fork and suggest changes.

Three steps!

Sample HTML

<span class="lighthouse image">
    <a href="full_image.jpg"><img src="small_image.jpg" /></a>
    <span class="caption">
        Caption
    </span>
</span>

Add the css

<link rel="stylesheet" href="jquery.lighthouse.css" />

Add the script (after jquery and jqueryui are loaded)

<script src="jquery.lighthouse.js"></script>

Use it!

<script>
   $('.lighthouse').lighthouse();
</script>

Options

Lighthouse supports setting custom options in the following syntax:

$('.lighthouse').lighthouse({
   option1: 'value1',
   option2: 'value2'
});

Defaults

How to find the container

containerSelector: '.lighthouse'

How to find the thumbnail anchor

anchorSelector: 'a'

How to find the thumbnail

imageSelector: 'img'

How to find the child container (not needed if contentType is image)

childSelector: '.content'

How to find the child anchor (not needed if contentType is image)

childAnchorSelector: 'a'

How to find the child image (not needed if contentType is image)

childImageSelector: 'img'

What the class of the close anchor will be

closeSelector: '.close'

What the class of the background div will be

backgroundSelector: '.background'

Sets both the open and close duration

duration: 350

Sets how long the open animation lasts

openDuration: null

Sets how long the close animation lasts

closeDuration: null

Sets how long any other animation lasts (background/close button/content fade in/out)

secondaryDuration: 200

What type of content determines whether or not a child div named content will be created (ie if it is an image then a special div will be created)

contentType: 'image'
